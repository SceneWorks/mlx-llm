## Summary

mlx-llm is a Rust project that implements a local LLM inference server targeting Apple hardware via the MLX array framework. It has two main technical components: (1) an HTTP server that exposes an OpenAI-compatible chat completion API over TCP, allowing clients to interact with locally-running language models using the same interface as the OpenAI API; and (2) a numerical accuracy test suite called sdpa_f32_repro that reproduces and validates scaled dot-product attention (SDPA) and matrix multiplication operations across different data types (e.g., f32, f16, bf16) to detect precision regressions in the MLX backend. The two components are coupled in that the test suite validates the correctness of the core attention computations that the inference server relies on. The project depends on MLX as its primary compute backend and targets Apple Silicon hardware where MLX is optimized to run. No external projects are known to depend on this repository.
