# mlx-llm

Reserved crate name for an upcoming **on-device text + vision LLM serving engine** built on Apple [MLX](https://github.com/ml-explore/mlx).

This `0.0.0` release is a placeholder — the implementation is in progress. A sibling [`candle-llm`](https://github.com/SceneWorks/candle-llm) crate will provide a Candle backend behind the same backend-neutral serving contract.

## Status

Work in progress. Not yet usable.
